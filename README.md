# Автотесты для формы заказа карты

## Описание проекта
Автотесты для тестирования формы заказа дебетовой карты с использованием Selenium.

## Технологии
- Java 11
- JUnit 5
- Selenium WebDriver
- WebDriverManager
- Gradle
